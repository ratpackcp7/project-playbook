# Task 001: Project Scaffold

## Objective

Create the project directory structure, virtual environment, dependency files, and configuration.

## Context

Starting from an empty project directory. SPEC.md has been completed with the full file tree and tech stack.

## Spec Reference

- File Tree (full directory layout)
- Tech Stack (language, framework, database, key packages)
- Environment (env vars, ports, config)

## Deliverables

- [ ] Complete directory structure matching SPEC.md File Tree
- [ ] `requirements.txt` with pinned versions from Tech Stack
- [ ] `.env.example` with all variables from Environment (no real values)
- [ ] `README.md` with project name, description, and setup instructions
- [ ] Virtual environment created and dependencies installed

## Acceptance Criteria

1. [ ] Directory tree matches SPEC.md exactly (no extra files, no missing files)
2. [ ] `pip install -r requirements.txt` succeeds in venv
3. [ ] `.env.example` contains every variable listed in SPEC.md Environment

## Notes

This is always the first task. No business logic — just structure and plumbing.
